Agile workflow

Je reste concentré sur les 3 philosophies du webdesign moderne, basé sur les bonnes pratiques de Google.

Mobile first

Des sites web qui atteignent plus de clients.

Avec plus d'un milliard d'appareils dans le monde, il est important de prioriser ses efforts sur cette population grandissante.

Responsive design

Des sites internet pour tous les appareils.

Une conception fluide permet à votre site internet de répondre aux besoins de vos utilisateurs en s'adaptant à leur largeur d'écran. Avec de nouveaux appareils disponibles chaque jour, un site responsive est une nécessité.

Performance

Un chargement rapide est indispensable.

La plupart des consommateurs s'attendent à avoir une page web chargée en moins de 2 secondes. Près de la moitié d'eux va abandonner un site web mettant plus de 3 secondes à se charger.

Projets open source

Je participe à l’intelligence collective du mouvement open source en partageant mes connaissances

  • Guide CSS

    Pendant des années j’ai travaillé sur des projets où le CSS peut être comparé à un bidonville : Un espace où s’entassent divers éléments, dans un chaos le plus complet. J’ai rédigé ce guide qui regroupe les bonnes pratiques pour coder des feuilles de styles de qualité, évolutives et maintenables.

  • Foundation Boilerplate

    Le point le plus important pour la création de nouveaux projets est de créer un ensemble de règles directrices. Foundation Boilerplate aide les designers & les développeurs à construire rapidement des prototypes d’application ou des sites web responsive évolutifs.

  • Refonte de site internet

    De nombreux sites web ne sont pas préparés pour le nouveau monde du multi-écran.
    Ce guide présente les différentes étapes pour transformer votre site web.
